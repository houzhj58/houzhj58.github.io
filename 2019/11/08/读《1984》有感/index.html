    读《1984》有感

    读书 1984 反乌托邦
    Created At : 2019-11-08 10:36
    Views 👀 :

    之前从各种渠道听闻了“老大哥”一词，知道出自于英国作家乔治奥威尔的《1984》一书。今天终于读完了。
    想起之前听一个朋友说，他的研究生面试时，老师提问了这样一个问题“你如何应对[The big brother is watching you]的情况？”，他当时不知道“the big brother”意味着什么，所以坦诚地说，不知道。
    那如果是我呢？我在那种情形下，会选择如何做呢？
    我家里有电幕，街上随处可能有隐藏麦克风，思想警察可能就在秘密监视着你，你吃饭时的服务员，购物时店里的老板，都可能是便装的思想警察。
    他们何止不允许你犯罪，想都不行，你任何下意识流露出的反动思想，都会成为逮捕你的导火索。你的子女被教导“要勇于揭发亲人的反动行为”。凡此种种，这样一种情形。简直压抑到极致。
    我会选择怎么办？
    我可能会选择成为一个无产者，书中的统治阶级认为，无产者是不足为惧的，只要思想控制得当，他们就是你可以操控的乌合之众，所以在无产者的生活环境中，没有那么严密的监视，这样才能以退为进，但这也可能只是为保全自我的无奈之举。
    书中男主人公认为“希望是存在于无产者中的”，只有无产者思想的到觉醒，才有可能推翻这样的统治，这也是为何统治阶级要创造新话运用各种手段来麻痹人们群众的原因，用仇恨、狂喜这种最直接的刺激，来保持人们的精神亢奋，不会去深思，而只是发泄。
    如果说“希望是存在于无产阶级中的”，那么势必由先进的思想去启迪无产者、带领无产者，那么在推翻当前统治阶级的进程中，是否也会用仇恨、狂喜、“人人平等”的愿景来麻木人群呢？
    这样想下去可能会有所悲观，但可悲的是，事实发生了验证了这个政治预言。我们看到了别人想让我们看到的东西，一些势力控制了资讯媒体，“美好的愿景”说给无产阶级听，适当范围内的改善作为“奶头乐”让我们停止哭闹。
    我们，人人平等，自由，人权，也许将来会实现，但不是现在。
